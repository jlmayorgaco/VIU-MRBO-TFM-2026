\section{Resultados esperados}

Se espera obtener un entorno de simulación organizado y reproducible que permita comparar, bajo un conjunto acotado de escenarios, el desempeño de un controlador nominal frente a otro adaptativo o robusto. El resultado principal esperado no es demostrar superioridad universal, sino identificar condiciones en las que la incorporación de adaptación frente a incertidumbre inercial aporta mejoras consistentes en seguimiento y robustez.

Asimismo, se espera generar evidencia gráfica y tabular suficiente para discutir compromisos entre precisión, esfuerzo de control y sensibilidad a degradación de la comunicación. Como producto académico complementario, el repositorio deberá dejar una base clara para ampliar el estudio en futuras iteraciones.
