\section{Objetivos}

\textbf{Objetivo general.} Diseñar y evaluar en simulación un marco de control distribuido para transporte cooperativo multi-AGV con incertidumbre inercial de la carga.

\textbf{Objetivos específicos.}
\begin{enumerate}
  \item Definir un modelo base bidimensional del sistema cooperativo con entre dos y cuatro AGVs, carga compartida y grafos de comunicación locales.
  \item Formular un controlador nominal distribuido basado en consenso como referencia comparativa.
  \item Proponer una extensión adaptativa o robusta frente a incertidumbre inercial de la carga sin exigir identificación exacta de todos los parámetros físicos.
  \item Establecer una batería de escenarios de simulación reproducibles para variación de masa, desplazamiento del centro de masas, degradación de comunicación y perturbaciones externas.
  \item Comparar cuantitativamente ambas estrategias mediante métricas de error de trayectoria, error de formación, esfuerzo de control, convergencia y robustez.
\end{enumerate}
