# Experimentos

La carpeta `experiments/` agrupa configuraciones reproducibles para campañas de simulación. Cada experimento dispone de un `config.yaml` autocontenido y un `README.md` breve que documenta su intención.

Experimentos iniciales:

- `exp-001-baseline-nominal`: línea base nominal.
- `exp-002-mass-variation`: sensibilidad a variaciones de masa.
- `exp-003-center-of-mass-shift`: desplazamiento del centro de masas.
- `exp-004-communication-degradation`: degradación del grafo de comunicación.
- `exp-005-disturbance-robustness`: perturbaciones externas y robustez.
