# Resultados

La carpeta `results/` separa artefactos generados por nivel de procesamiento:

- `raw/`: salidas directas de ejecuciones.
- `processed/`: datos derivados o agregados.
- `figures/`: gráficas generadas.
- `tables/`: tablas listas para informes.
- `reports/`: resúmenes exportables.

Los subdirectorios incluyen `.gitkeep` para preservar la estructura sin versionar resultados transitorios.
